# 00. PRD Core: 몬스터즈 계산기 (Monsters Calculator) 핵심 기획

## 1. 프로젝트 개요
React Native(Expo)로 제작하는 가벼운 캐릭터 테마 계산기 앱이다. 
유틸리티 앱 생태계(계산기, 날씨, 다이어리 등)의 첫 번째 앱으로, 공통된 UX와 캐릭터 세계관을 제공한다.
핵심 가치: 계산 과정의 스트레스를 줄여주는 '감성적 완충재' 역할을 하는 캐릭터 "블루우", 그리고 생활에 밀접한 '빠른 계산' 제공.

- **앱 이름:** 몬스터즈 계산기 (Monsters Calculator)
- **메인 캐릭터:** 블루우 (Bluu)

## 2. 앱의 핵심 경험
1. **감성적 완충재:** 복잡한 계산이나 에러 발생 시, 딱딱한 시스템 메시지 대신 캐릭터 '블루우'가 당황하거나 헷갈려하는 반응을 보여줌으로써 사용자의 스트레스를 줄인다. 캐릭터 애니메이션은 비동기로 처리되어 계산기 본연의 빠른 사용성을 절대 방해하지 않는다.
2. **빠른 생활 계산:** 일상에서 자주 쓰는 계산(더치페이, 할인, 팁, 퍼센트)을 카드 형태로 빠르게 제공한다.

## 3. 캐릭터 '블루우' 컨셉 및 운용 원칙
- **정체성:** 숫자를 좋아하는 작은 파란색 몬스터 친구.
- **배치 원칙:** 메인 화면의 시선 분산을 막기 위해 **결과창 위(또는 키패드 '0' 버튼 등)** 단일 위치에만 등장한다. 헤더 영역에는 캐릭터를 배치하지 않는다.
- **반응 원칙:** 사용자 입력을 막지 않도록(Non-blocking) 비동기로 부드럽게 반응한다.

## 4. 앱 네비게이션 구조 (Bottom Tabs)
- **Tab 1. 계산기 홈:** 기본 사칙연산 계산기 (캐릭터 '블루우' 상주)
- **Tab 2. 빠른 계산:** 4가지 상황별 계산 카드 리스트
