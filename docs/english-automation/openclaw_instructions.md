## 🔄 매일 수행할 작업 (Step-by-Step)

### Step 0: 오늘의 학습 찾기

1. `daily_schedule.csv`를 읽는다.
2. 오늘 날짜(`YYYY-MM-DD`)에 해당하는 행을 찾는다.
3. 해당 행이 없으면 → "오늘은 학습 스케줄이 없습니다." 로그를 남기고 종료.
4. 해당 행에서 다음 정보를 추출한다:
   - `day`: 학습 일차
   - `episode`: 에피소드 번호
   - `video_id`: YouTube 영상 ID
   - `start_seconds`: 시작 시간 (초)
   - `end_seconds`: 종료 시간 (초)
   - `segment_label`: 세그먼트 설명

### Step 1: YouTube 트랜스크립트 가져오기

**MCP 도구 사용:** `youtube-transcript__get_transcript`

1. `video_id`를 사용하여 해당 YouTube 영상의 **영어 자막(transcript)**을 가져온다.
2. 가져온 전체 트랜스크립트에서 `start_seconds` ~ `end_seconds` 범위에 해당하는 부분만 추출한다.
3. 추출된 텍스트를 **깔끔하게 정리**한다.
4. 출력 시에는 반드시 **영어 원문을 먼저 충분히 보여주고**, 그 다음에 **같은 범위의 한국어 해석을 따로 이어서** 보여준다.

### Step 2: 핵심 스크립트 확장 정리

추출된 스크립트에서 **문맥 이해와 암기에 도움이 되는 핵심 스크립트**를 충분히 선정한다.

**작성 원칙:**

- 짧은 한 문장만 고르지 말고, 필요하면 **1~3문장 묶음**으로 잡는다.
- 최소 **12개 이상** 작성하고, 학습 가치가 높으면 더 많이 작성한다.
- 먼저 **핵심 스크립트 영어 모음**을 한 번에 보여주고, 그 다음에 **같은 순서의 한국어 해석 모음**을 보여준다.
- 표현만 뽑는 것이 아니라, 앞뒤 문맥이 살아 있도록 구성한다.

### Step 3: 중요 문장 20개 선정

추출된 스크립트에서 **영어 학습에 가장 유용한 문장 20개**를 선정한다.

**선정 기준:**

- 일상 대화에서 자주 쓰이는 표현이 포함된 문장
- 관용적 표현(idiom), 구동사(phrasal verb), 슬랭(slang)이 포함된 문장
- 문법적으로 배울 점이 있는 문장 (가정법, 도치, 관계절 등)
- Tablo의 위트 있는 표현이나 독특한 비유가 담긴 문장
- 원어민이 실제 사용하는 자연스러운 구어체 문장

**출력 원칙:**

- 먼저 **핵심 문장 20개 영어 원문**을 1번부터 20번까지 모두 보여준다.
- 그 다음에 **같은 순서의 한국어 해석**을 1번부터 20번까지 이어서 보여준다.
- 각 문장에는 짧게 **왜 중요한지**도 덧붙인다.

### Step 4: 핵심 단어 & 표현 정리

선정된 20개 문장에서 **학습이 필요한 핵심 단어/표현**을 추출하고 한국어로 설명한다.

**각 단어/표현에 포함할 내용:**

- 🔤 **영어 단어/표현**: 원문 그대로
- 📖 **뜻 (한국어)**: 한국어 의미
- 💬 **사용 예시**: 해당 문장에서 어떻게 쓰였는지
- 📝 **추가 설명**: 뉘앙스, 유의어/반의어, 사용 상황 등

## ✅ Step 5: 진행 상황 기록

학습 자료 생성이 완료되면 `study_progress.csv`에 다음 행을 추가한다:

```
{day},{date},{episode},{segment_label},completed,output/dayXX_epY_partZ.md,자동 생성 완료
```

---

## ⚠️ 에러 처리

- **트랜스크립트를 가져올 수 없는 경우**: `study_progress.csv`에 `status=failed`로 기록하고, `notes`에 에러 내용을 남긴다.
- **스케줄에 오늘 날짜가 없는 경우**: 아무 작업도 하지 않는다.
- **이미 완료된 학습인 경우**: `study_progress.csv`를 확인하여 이미 해당 날짜가 `completed`로 기록되어 있으면 스킵한다.

## 📌 참고사항

- 모든 출력 파일은 **영어 원문 먼저, 그 다음 한국어 해석** 순서를 기본으로 합니다.
- 전체 스크립트와 핵심 스크립트는 너무 짧게 요약하지 말고, **공부에 충분할 정도로 넉넉하게** 작성합니다.
- 문장 선정 시 **실용성**을 최우선으로 합니다 (시험 영어 X, 실생활 영어 O).
- 한국어 및 영어 외의 언어가 있으면 다시 작성합니다.
