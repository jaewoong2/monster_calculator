# 02. PRD Quick Calc: 빠른 계산 탭

## 1. 화면 목표
일상생활에서 자주 필요하지만 일반 계산기로는 여러 번 눌러야 하는 계산을 '카드 형태'로 빠르고 쉽게 해결한다.

## 2. 빠른 계산 카드 리스트 (4종)
모든 카드는 내부 로직(입력 필드 2개 -> 결과 도출)이 유사하므로 컴포넌트를 재사용하여 구현한다.

### 1) 나눠 계산하기 (더치페이)
- **입력:** 총 금액, 인원수
- **결과:** 1인당 금액 (남는 금액 안내 포함)
- **캐릭터:** 사람 아이콘을 가리키는 포즈

### 2) 할인 계산하기
- **입력:** 할인 전 원가, 할인율(%)
- **결과:** 할인된 금액, 최종 결제 금액
- **캐릭터:** 할인 태그(%)를 들고 있는 포즈

### 3) 팁 계산하기
- **입력:** 총 금액, 팁 비율(%)
- **결과:** 팁 금액, 최종 결제 금액
- **캐릭터:** 동전이나 엄지척 포즈

### 4) 퍼센트 계산
- **입력:** 기준 값, 비율(%)
- **결과:** 해당 비율의 값 (예: 50,000원의 15%)
- **캐릭터:** 돋보기 포즈

## 3. UX/UI 원칙
- 각 기능은 카드명만 봐도 직관적으로 이해되어야 함.
- 입력 필드는 최소화(2~3개)하고, 결과를 바로 확인할 수 있어야 함.
